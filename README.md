# E-Commerce-Analytics-Project
End-to-End Business Intelligence Portfolio Project

Tools: Excel · Power BI · Tableau · SQL

Project Overview
This end-to-end analytics project transforms raw e-commerce transactional data into a fully interactive Power BI Dashboard that surfaces revenue trends, customer behavior, product performance, and logistics efficiency — enabling data-driven decisions across marketing, operations, and supply chain.

Business Problem

An e-commerce company with 1B+ in revenue across multiple product categories lacked a centralized view of:
Which categories & products drive revenue
How customer demographics (gender, age group, geography) affect purchasing
Whether shipping methods impact cost and customer satisfaction
Which products are at risk of stock-outs or underperformance

  Key Features


🔍 Seasonality Filter — toggle Yes/No to compare seasonal vs non-seasonal demand
🗂️ Category Slicer — drill down into Books, Electronics, Footwear, Home Appliances, Apparel
📍 Customer Location Filter — geographic segmentation across all visuals
📱 Cross-filtering — all visuals interact with each other dynamically
📈 Supplier & Stock Level Table — operational view for inventory teams
